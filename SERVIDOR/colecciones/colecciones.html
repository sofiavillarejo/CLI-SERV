Colecciones

Listas

Una lista puede contener varios elementos:

["manzana", "platano", "naranja"]

Están ordenadas y empiezan en el índice 0

Se pueden cambiar, añadir y borrar elementos de la lista

Permite elementos duplicados.

Para saber el tamaño de una lista se usa la función len(nombre_de_la_lista)

Se pueden mezclar tipos de datos en los elementos de una lista

La función type() devuelve class "list"

Tuplas

Una tupla puede contener varios elementos

("manzana","platano","naranja")

Están ordenadas y empiezan en el índice 0

No se pueden modificar, añadir ni borrar elementos.

Permite elementos duplicados.

Para saber el tamaño de una tupla se usa la función len(nombre_de_la_lista)

Se pueden mezclar tipos de datos en los elementos de una tupla

La función type() devuelve class "tuple"

Diccionarios

Almacén de información en formato clave:valor

{"clave1":"valor1", "clave2":"valor2", "clave3":"valor3"}

Son ordenados

Se pueden cambiar, añadir y borrar elementos de la lista

No permite claves duplicadas

Se pueden mezclar tipos de datos en los valores de un diccionario. Las claves son str

La función type() devuelve class "dict"

Conjuntos

Un conjunto puede contener varios elementos

{"manzana","platano","naranja"}

No están ordenadas

No se pueden modificar, pero SI añadir y borrar elementos.

No permite elementos duplicados

True y 1 son lo mismo

Se pueden mezclar tipos de datos en los elementos de un conjunto

La función type() devuelve class "set"

Si queremos añadir elementos que ya están en el conjunto, no se añaden.
